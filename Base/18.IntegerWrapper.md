
### Java 8 種原始資料型別

- 整數whole number：byte、short、int、long
    
- 浮點數 real numbe(floating point or decimal)：float、double
    
- single character：char

- boolean value : boolean  

原始型別是記憶體中存放資料的基本單位
    
---

## int 的特性

- 整數，不含小數
    
- 有固定可用範圍
    
    - **最小值：** `Integer.MIN_VALUE` → -2,147,483,648
        
    - **最大值：** `Integer.MAX_VALUE` → 2,147,483,647
        
- 超出範圍會溢位或下溢（wraparound）
    

---

## Wrapper Class（包裝類別）

- Java 對其八種原始資料型別都採用「包裝類別（wrapper class）」的概念
    包裝類別提供一些簡單的操作方法，以及原始型別本身無法儲存的基本資訊

- 每個 primitive 都有對應 wrapper，例如：
包裝類別名稱通常只是把原始型別的名稱首字母大寫。不過有兩個例外：char 對應 Character、int 對應 Integer
    
byte ---> Byte

short ---> Short

int ---> **Integer**

long ---> Long

float ---> Float

double ---> Double

char ---> **Character**

boolean ---> Boolean

- Wrapper 提供額外功能（如 MIN_VALUE、MAX_VALUE）
    

---

## Overflow / Underflow（溢位/下溢）

- `Integer.MAX_VALUE + 1` → 變成最小值（溢位）
    
- `Integer.MIN_VALUE - 1` → 變成最大值（下溢）
    
- 整數回繞（wraparound）可能導致錯誤結果  
    （例如計數器超過 21 億後變成負數）
    

---

## 字面值 vs 運算式超範圍

- 字面值太大 → 編譯器報錯  
    `int x = 2147483648; // error`
    
- 運算式結果太大 → 不報錯，但溢位  
    `int y = Integer.MAX_VALUE + 1; // wraparound`
    

---

## 字串串接輸出

- 使用 `+` 來接字串＋數字  
    `System.out.print("Min = " + Integer.MIN_VALUE);`
    

---

## 數字底線 _ 提升可讀性

- 可用：`2_147_483_647`
    
- 不可用在開頭或結尾
    
- 視覺上類似加逗號，但語法合法
