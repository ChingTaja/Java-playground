Postfix Increment
```java
result++;  // 等同於 result = result + 1
```
- 不需要再寫 =
- 只針對變數加 1

Postfix Decrement
```java
result--;  // 等同於 result = result - 1
```
- 不需要再寫 =
- 只針對變數減 1

| 運算子  | 等同於這樣的寫法               |
| ---- | ---------------------- |
| `+=` | `x += y` → `x = x + y` |
| `-=` | `x -= y` → `x = x - y` |
| `*=` | `x *= y` → `x = x * y` |
| `/=` | `x /= y` → `x = x / y` |


 報錯： 資料型別不相容
```java
int result = 10
result = result - 5.5
system.out.print(result)
```
改為 casting
```java
int result = 10;
result = (int)(result - 5.5); // 結果會變成 4
System.out.println(result);
```
1. Java 會先將 int(result) → double 進行運算（自動提升型別）
-> 所以 10 - 5.5 = 4.5

2. 這個運算結果是 double
(int)(result - 5.5)
(int) 是 強制轉型（cast）
將 double 4.5 轉成 int

轉型規則：小數部分直接捨棄（不是四捨五入）
所以 4.5 → 4

result -=5.5;
result = (int) (result - 5.5)
