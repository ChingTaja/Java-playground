浮點數（floating point numbers）浮點數包含小數部分 , 使用 小數點（.） 表示
浮點數也被稱為 實數（real numbers）

計算中需要 `更高的精確度` 時，就會使用浮點數

用來表示浮點數的基本資料型別有兩種：
`float` 和 `double`

double 是 Java 中所有小數或實數的`預設`型別

任何帶有小數的數字在 Java 中預設都是 double

所以，10.5 在 Java 中預設就是 double

double 型別的數值常值（numeric literal）可以加上後綴 `小寫 d 或 大寫 D `來明確指定
但因為在 Java 中 double 本身就是預設型別
這個後綴是可選的

float 型別的數值常值則必須加上後綴 `小寫 f 或 大寫 F`
如果你要`把一個實數（real number）指定給 已宣告為 float 的變數，這個後綴就是必須的`

重要的一點是，Java 並`沒有`選擇 較小、精度較低的 float 型別
原因是：使用`更精確的型別通常比節省記憶體更重要`

```java
float floatValue = 2.5 // error ! float 的精度比 double 低，所以你不能直接把精度較高的 double
指派給精度較低的 float 變數

float floatValue = 2.5f
```
