# char
`char` 型別：這個字元可以是任何單一字元

char 本質上存的是「數字」，只是 Java 幫你把那個數字「翻譯成字元」給你看

```java
char myChar = 'D';
```
--> 你以為裡面存的是字母 D，
但實際上 Java 存的是一個數字：68。
因為在 Unicode 裡：'D' = 68

很重要的一點是：char 和 String 是不同的型別
char 在某種程度上和 String 類似，因為它們都可以儲存字元

但 char 只能儲存一個字元
String 可以儲存多個字元

char 的常值用單引號
String 則用雙引號

你可能會開始想：
「這種型別到底有什麼用？`為什麼`要用只能存一個字元的變數？」

以下是一些實際例子：
1. 儲存遊戲中「使用者最後按下的按鍵」
2. 以程式方式逐一處理英文字母（例如 A 到 Z）

之後會研究迴圈（loop）和陣列（array），到時會更清楚 char 的實際用途

char 是以 2-byte 的數值形式 儲存

## 什麼是 Unicode？

Unicode 是一種國際編碼標準
為每個字母、數字或符號分配一個唯一的數值
可在不同平台與程式間通用

英文字母只需要 26 個字元（A–Z）
但其他語言需要的字元數量多得多

Unicode 就是為了`支援這些語言`而存在的

由於 char 佔 2 個位元組
它最多可以表示 65,535 種不同的字元

## 在 Java 中使用 Unicode
```java
char myUnicode = '\u0044';
```
